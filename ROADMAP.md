# Roadmap

## 未來規劃 (Future Plans)

### 階段一：基礎建設與自動化 (v1.0 - v1.1) ✅
- [x] **雲端託管**: 支援 Render 部署與 24/7 在線。
- [x] **自動週報**: 每週自動統計並發布報告。
- [x] **靈活統計**: 支援月報與自訂日期範圍。
- [x] **文件發布**: 支援透過指令發布更新公告與說明文件。

### 階段二：資料持久化與網頁儀表板 (v1.2 - v1.3) 🚧
- [ ] **資料庫整合**: 將數據從記憶體/JSON 遷移至資料庫 (如 MongoDB 或 PostgreSQL)，確保資料不遺失。
- [ ] **網頁儀表板 (Web Dashboard)**: 讓學生與老師可以透過網頁查看詳細的個人數據與圖表。
- [ ] **歷史數據查詢**: 支援查詢過去任何一週的歷史紀錄。

### 階段三：進階互動與 gamification (v2.0) 🔮
- [ ] **經驗值系統 (XP)**: 更細緻的經驗值計算與等級系統。
- [ ] **商店與道具**: 使用積分兌換虛擬獎勵或 Discord 身份組。
- [ ] **跨平台整合**: 與 OCW 課程網站進行資料連動。
