# Release Note: v1.1.0 Online

## 📅 更靈活的時間統計

這個版本帶來了大家期待已久的**自訂時間範圍**功能！現在您不再受限於「週」報，可以自由查詢月報或任意時段的成績。

### ✨ 新增功能
1.  **月報與自訂範圍查詢**
    - `/compute month:11`: 直接計算 11 月整個月的成績。
    - `/compute start_date:2025-11-01 end_date:2025-11-15`: 計算指定半個月的成績。
    - 當然，`/compute week:45` 仍然可以使用。

2.  **清晰的日期標示**
    - 所有的指令 (`/leaderboard`, `/matrix`, `/attendance` 等) 現在都會在標題顯示當前的統計範圍。
    - 例如：`🏆 排行榜 (Month 11 | 2025-11-01 ~ 2025-11-30)`
    - 讓您清楚知道現在看到的數據是哪一段時間的。

### 🛠️ 升級說明
- **Render 用戶**: 程式碼推送到 GitHub 後，Render 會自動部署新版本 (v1.1.0)。
- **設定**: 無需更改任何環境變數。
