# Release Note: v1.0 Online

## 🚀 重大更新：雲端託管與自動化

此版本是專為 **24/7 全天候運作** 所設計的重大更新。我們將機器人從本地執行遷移至雲端環境 (Render)，並強化了安全性與自動化功能。

### ✨ 新增功能
1.  **自動週報系統**
    - 機器人現在會**每週一凌晨 00:00 (UTC+8)** 自動醒來。
    - 統計上一週 (週一至週日) 的學生活躍度。
    - 自動將「統計結果」與「排行榜」發送到指定的公告頻道。
    - 再也不用手動輸入指令了！

2.  **雲端託管支援 (Render)**
    - 新增 `keep_alive.py` 與 `render.yaml`。
    - 讓機器人可以免費部署在 Render 平台上。
    - 配合 UptimeRobot 可達成永不休眠。

3.  **安全性升級**
    - 移除程式碼中的 Token 與 ID。
    - 改用 **環境變數 (.env)** 管理敏感資訊，開源更安心。

4.  **全伺服器動態追蹤**
    - 不再需要手動維護學生名單 (`USER_IDS`)。
    - 機器人會自動抓取伺服器內的所有成員進行統計。

### 🛠️ 如何升級
如果您是從 v1.3 升級：
1. 請確保您已設定好 `.env` 檔案 (參考 `DEPLOY_GUIDE.md`)。
2. 將程式碼部署至 Render。
3. 設定 Render 的環境變數。
4. 享受自動化的便利！
